@vangware/micro

@vangware/micro

This library is a collection of functional utils made entirely in TypeScript. Is the successor of @vangware/micron, with less focus on small footprint and higher focus on utility and readability across all JS environments (Node and the browser).

Installation

You can use npm, pnpm or yarn to install it locally:

npm i -S @vangware/micro

Or you can use it directly as an ESM module trough services like UNPKG:

import { isObject } from "https://unpkg.com/@vangware/micro?module";
